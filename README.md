# Algorithm_ABC_salesman-s-problem
Application of the bee colony algorithm to find the shortest route between points to solve the traveling salesman problem



Instructions for use: 
1. Create a folder for the project
2. Copy the 3 files into the folder:
  - tsp_abc.cpp
  - tsp_solver.py
  - README.md
3. Install dependencies: pip install numpy matplotlib
4. Run: python tsp_solver.py
